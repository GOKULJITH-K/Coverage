Functional Coverage Projects

This site documents a collection of functional coverage–driven verification projects, focused on validating protocol behavior using SystemVerilog covergroups, assertions, and simulation-based analysis.

Each project follows a feature-oriented verification methodology, with test-case–wise coverage collection and interactive HTML dashboards.

Verification Methodology

The following principles are consistently applied across all projects:

  • Feature-based functional coverage
  • Directed and constrained-random stimulus
  • Assertion-based protocol checking
  • Per-test-case coverage dashboards generated using Synopsys URG
